Error Rate Metrics & Harvest Recall Specs

We distinguish between character accuracy, capture miss rate, and pipeline loss rate.

CER
Character Error Rate

$$\text{CER} = \frac{\text{Substitutions} + \text{Deletions} + \text{Insertions}}{\text{Ground-Truth Characters}}$$

Pmiss
Capture Miss Rate

$$P_{\text{miss}} \approx \max(0, 1 - f_{\text{effective}} T)$$

100%
Harvest Recall Rate

$$\text{Harvest Recall} = \frac{\text{Lines Recovered}}{\text{Lines Displayed}}$$
